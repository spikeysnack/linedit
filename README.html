linedit -- a one line editor

1.2 12 Jul 2017 beta Free for all non-commercial purposes. Copyright 2017 Chris Reid spikeysnack@gmail.com

Usage: linedit 

    information options:

-c   print line in diagnostic mode
        -h   show this helpful message
        -i   show instructions
        -l   show licensing info
        -q   do not show results
        -v   show version

        editing options:

        -a   append  a line to file
        -p   prepend a line to file

    Example:

         `linedit myfile.txt 15`

Instructions

linedit <filename> <line#> linedit -c <filename> <line#> linedit <filename> linedit -a <filename> linedit -p <filename> linedit -h

Type the filename and line number to edit.

The line will be echoed in light blue.

The line text will be placed after an edit: prompt.

Navigate with keys and edit.

Type to change the line in the file.

Type - to abandon the edit.

If the line is not changed, no change is printed.

If the line is changed, a visual representation of the line is printed with spaces, tabs, strings, and escaped characters highlighted in color. The previous line is saved to .le with a timestamp.

A multi-line grep command is output to show line and two lines before and after.

If is 0 the line will be prepended.

If is the edited line will be appended.
